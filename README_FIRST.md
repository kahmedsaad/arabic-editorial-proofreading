# Cursor Starter Bundle

## Included
- PROJECT_CONTEXT.md: complete backend-engine context
- prompts/: staged Cursor prompts
- GEMINI_SYSTEM_PROMPT.txt: model system prompt
- docs/AI_Proofreading_Agent_Proposal_V2.pdf: reference proposal
- data/pairs.zip: provided before/after Arabic pairs

## How to use
1. Create a new empty folder or Git repository in Cursor.
2. Copy this bundle into the repository root.
3. Open `PROJECT_CONTEXT.md` in Cursor and pin it to context.
4. Run the prompt in `prompts/01_INITIAL_BUILD_PROMPT.txt` first.
5. Review and test the result before moving to prompt 02.
6. Add Gemini only after the local mock pipeline is stable.
